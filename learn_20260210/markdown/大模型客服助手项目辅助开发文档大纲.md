# 大模型客服助手项目辅助开发文档大纲

#  一、项目概述（供管理层理解）

## 1.1 项目背景

- 传统客服模式痛点分析

- 大模型技术在客服领域的应用价值

- 苏商银行远程客服中心现状与挑战

## 1.2 项目目标

- 提升客服自助解决率

- 增强在线客服并行会话能力

- 实现全流程智能化质检

- 降低人力成本，提升客户满意度

## 1.3 项目创新点
- 客服知识库助手：自动化知识管理

- 话术推荐助手：RAG实时推荐

- 质检助手：全自动化情感与专业性质检

# 二、系统架构总览
## 2.1 整体架构图（附说明）

- 三层架构：应用系统 → 大模型系统 → 平台模型 → 支撑系统

- 各层功能介绍

## 2.2 子系统定位与职责

- **员工助手子系统**：提升员工效率，支持多种场景

- **业务智能子系统**：为业务系统提供智能服务

- **RAG服务子系统**：提供检索增强生成能力

- **大模型服务子系统**：提供多模态处理能力

- **大模型管理子系统**：提供配置与管理后台

# 三、核心功能模块（开发重点）
## 3.1 客服知识库助手

- 业务流程梳理

- 功能清单：

    - 客户问题抽取

    - 客户意图识别与改写

    - 客服标准问/相似问生成

    - 电话话术提炼

    - 电催话术提炼

## 3.2 话术推荐助手

- 业务流程梳理

- 功能清单：

    - 实时对话情境理解

    - 用户意图识别

    - RAG话术推荐

    - 个性化话术适配

## 3.3 质检助手

- 业务流程梳理

- 功能清单：

    - 对话内容上传与解析

    - 语音转文字处理

    - 大模型自动化质检

    - 结果生成与复核流程
